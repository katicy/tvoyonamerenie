# твоё намерение

Сайт-сообщество про манифестацию: лента намерений, форум, дневник аффирмаций.

## Стек
Next.js 14 + Supabase (база данных и авторизация) + Vercel (хостинг)

## Как запустить (см. пошаговую инструкцию в чате с Claude)
1. Создать проект в Supabase, выполнить `supabase-schema.sql` в SQL Editor
2. Скопировать `.env.local.example` в `.env.local` и вставить свои ключи из Supabase
3. Залить код на GitHub
4. Подключить репозиторий к Vercel, добавить те же переменные окружения
5. Deploy
